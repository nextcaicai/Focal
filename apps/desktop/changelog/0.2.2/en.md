# What's new in v0.2.2

## New Features

- Update notification system: replaced the legacy update notice with `UpdateVersionBadge` and a shared update state hook, so available versions are surfaced in Settings and the subscription column.
- YouTube playback controls in the Videos layout: pin the player, auto-scroll the transcript, and highlight the current line while watching.

## Improvements

- Richer update prompt indicators across the About tab and Settings for quicker discovery of new versions.
- Nested modal and dropdown handling: menus, selects, and context menus inside modals no longer dismiss the parent dialog accidentally.

## Bug Fixes

- Settings action buttons now use the accent color instead of red for visual consistency.
- Settings modal layout: improved responsiveness and overflow handling in the settings panel and background processing section.
- Fixed portaled select menus closing modals when choosing an option.

## Thanks

Thanks for using v0.2.2 and sharing your feedback.
