# v0.2.6 更新內容

## 新功能

- **全庫搜尋**：側邊欄新增搜尋入口（與今天 / 未讀 / 收藏並列），可跨全部訂閱檢索文章；結果依相關度、時間與品質分排序，並支援譯文標題比對。
- **語意搜尋**：啟用 Embedding 後，搜尋同時使用關鍵字與向量相似度——同義說法、中英換詞也能召回；搜尋標題下會顯示語意索引進度。
- **語意索引涵蓋已讀歷史**：Embedding 不只處理未讀，也會為已讀與歷史條目建立向量（與 LLM 摘要／打標分開，不額外消耗對話類 token）。
- **AI 標籤詞表升級（形態／領域／議題）**：標籤體系支援內容形態、領域與議題主題三軸；舊標籤會依對應表離線整理，避免為全庫重跑打標。

## 改進

- 關鍵字主題（My Topics）在具備向量時，會結合語意相似度比對相關文章，而不只看標題字面。
- 側邊欄文案由「查找」統一為「瀏覽」，導覽語意更一致。
- 無訂閱時空狀態改為直接進入 Discover 發現頁，移除臨時的簡易新增彈窗。
- Embedding 處理進度與重建說明改為涵蓋「全部可嵌入條目（含已讀）」，狀態更透明。
- 多處事件監聽與 Hook 依賴整理，介面互動更穩、重繪更乾淨。

## 問題修復

- Embedding 任務佇列去重：避免同一篇文章被反覆入隊導致「排隊」虛高、重複計算與介面卡頓。
- 庫搜尋工作階段狀態精簡，移除未使用的範圍／排序欄位，減少異常狀態殘留。
- 搜尋與列表相關在地化文案補齊與統一（中／英／日／繁中／法）。

## 致謝

感謝你使用 v0.2.6 並持續回饋——全庫搜尋與語意索引尤其需要真實資料庫來校正體驗。
