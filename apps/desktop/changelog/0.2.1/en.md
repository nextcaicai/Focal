# What's new in v0.2.1

## New Features

- Selected text context menu: send highlighted text to Obsidian, Outline, or Cubox.
- Multi-language in-app release notes: the "What's New" dialog now follows your app language.

## Improvements

- Replaced legacy Follow changelog files with Focal-owned release notes.
- Improved update prompts: brand-colored dot on the About tab, version badge on Check for Updates, and a compact version label beside the sidebar Settings button—tap to open the download page.

## Bug Fixes

- Fixed entry hydration limit issues during timeline loading.

## Thanks

Thanks for using v0.2.1 and sharing your feedback.
