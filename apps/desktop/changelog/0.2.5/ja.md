# v0.2.5 の新機能

## 新機能

- 記事翻訳の刷新：ストリーミング翻訳による段階的更新に対応し、**バイリンガル** / **訳文のみ** の表示切り替えを追加しました。
- ローカル RSS の履歴インポート：現在のフィードウィンドウから利用可能なエントリを取得できます（既読として取り込み、AI 処理なし）。新規・既存の購読とも、フィードが許可する範囲で履歴を補完できます。
- 記事ページでのオンデマンド AI 要約：必要なときに自分のキーで要約を生成。履歴全体への自動要約は行いません。

## 改善

- AI 設定の表記を **LLM モデル** / **Embedding モデル** に統一し、BYOK などの専門用語を避けました。空状態やエラー文言も用途が分かりやすくなっています。
- 要約・タグ・品質スコアなどの **AI 自動処理はデフォルトでオフ**。モデルを設定し、明示的に有効化するまで動きません。初期状態では静かなローカル RSS リーダーとして使えます。
- AI 設定のレイアウトとアクション説明を見直し、トークン消費と「新規未読のみ処理」の挙動を把握しやすくしました。
- Discover の導線を簡素化し、RSSHub 中心のルートと関連 UI を削除しました。
- エンリッチメントに自動リトライ上限と失敗トラッキングを追加し、失敗したフェーズが無限に回り続けるのを防ぎます。
- スマートフィードの「すべて既読」が日付範囲とリスト境界をより正しく扱います。
- リスト要約でバイリンガル／生成説明の選択を改善し、行数制限向けに Markdown を正規化しました。

## 修正

- 記事未選択時と空リストのプレースホルダー文言を分かりやすくしました（「記事を選んで読み始めましょう」「今は静かです」）。
- 翻訳表示スイッチャーの余白とラベルを整理しました。
