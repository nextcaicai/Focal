# v0.2.5 更新內容

## 新功能

- 文章翻譯大改版：支援串流翻譯與漸進更新，並提供 **雙語對照** / **僅譯文** 顯示切換。
- 本機 RSS 歷史匯入：可從目前訂閱源視窗拉取可用條目（匯入為已讀、不走 AI）。新訂閱與既有訂閱在來源允許時皆可補齊歷史。
- 文章頁依需求產生 AI 摘要：需要時用自己的金鑰產生，預設不會對整段歷史自動跑摘要。

## 改進

- AI 設定文案統一為 **LLM 模型** / **Embedding 模型**，不再使用難懂的 BYOK 說法；空狀態與錯誤提示更清楚說明各模型用途。
- 摘要、標籤、品質分等 **AI 自動處理預設關閉**，需你設定模型並主動開啟——開箱即用時更像安靜的本機 RSS 閱讀器。
- 調整 AI 設定版面與動作說明，更容易理解 token 消耗與「僅處理新未讀」的行為。
- 精簡 Discover 導覽，移除以 RSSHub 為中心的路由與相關介面路徑。
- 豐富化任務增加自動重試上限與失敗追蹤，避免失敗階段一直空轉。
- 智慧訂閱的「全部標為已讀」在日期範圍與列表邊界上更可靠。
- 列表摘要更合理地區分雙語／產生描述，並規範 Markdown 以便行數裁剪。

## 問題修復

- 未選中文章與空列表的佔位文案更清晰（「選擇一篇文章開始閱讀」「這裡暫時很安靜」）。
- 翻譯顯示切換器的間距與標籤整理得更簡潔。

## 致謝

感謝你使用 v0.2.5 並提供寶貴回饋。
